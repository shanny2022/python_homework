# Assignment 7 Help

Use `CTD_Assignment_7.ipynb` as your Kaggle notebook guide.

## Kaggle Steps

1. Create a new Kaggle notebook named `CTD_Assignment_7`.
2. Add the Kaggle dataset `Diabetes Health Indicators Dataset` by Alex Teboul.
3. Copy the cells from `CTD_Assignment_7.ipynb` into Kaggle, or upload the notebook file.
4. Run every cell from top to bottom.
5. Read the markdown interpretation cells and personalize the wording where you can.
6. Click `Save Version`, choose the default quick save, then use `Share` to make the notebook public.
7. Submit the public Kaggle URL.

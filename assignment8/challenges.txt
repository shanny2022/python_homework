Task 6: Challenges

The Durham County Library page is generated by a modern catalog system, so the search results are easier to understand in the browser developer tools than in the static page source. I handled this by using Selenium and several focused fallback selectors for the result item, title, author, and format/year fields.

The OWASP project page has changed to point to the current 2025 Top Ten release. I resolved this by starting at the assigned project page, using XPath to find the current release link, and then extracting the A01 through A10 links from the ordered list on that release page.

I also used headless browser options and limited the scraper to the required pages so the programs avoid unnecessary repeated requests.
